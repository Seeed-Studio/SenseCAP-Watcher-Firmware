# SenseCAP Watcher出厂固件

![watcherbanner](docs/img/watcherbanner.jpg)

SenseCAP Watcher是全球首个革命性的空间物理感知AI代理。这个项目是其出厂固件。

## 特性

- tinyML + LLMs（以及本地部署的LLM服务）
- 类似NODE-Red功能模块架构
- 基于工作流的任务执行
- 自然语言交互
- 高度交互式用户界面
- 可配置的SenseCraft App
- 固件OTA
- 更简便的模型训练
